# ワンルーム投資危険度チェッカー

営業トークではなく、データで判断するための無料Webアプリです。

## 概要

このアプリでは、次の2つを同じUIの中で確認できます。

- 危険度チェック: 利回り・相場比・ランニングコストから入口条件を確認
- 長期シミュレーション: ローン、家賃下落、空室、修繕、売却価格を踏まえて出口まで確認

シミュレーション画面では、楽観・標準・悲観のシナリオを初期値として適用しつつ、各数値を直接入力したりスライダーで微調整したりできます。

## 起動方法

```bash
npm install
npm run dev
```

## ビルド方法

```bash
npm run build
npm run preview
```

## GitHub Pages で公開する流れ

1. GitHub にリポジトリを作成
2. ソースを push
3. `npm run build` で `dist` を生成
4. GitHub Actions または `dist` 配信で GitHub Pages に公開

`vite.config.ts` は `base: './'` にしてあるため、静的配信でも詰まりにくい構成です。リポジトリ名固定で運用する場合は `/repo-name/` に変えてもOKです。

## 補足

- 相場データは現状MVP用サンプル値です
- 不動産情報ライブラリAPIに差し替えやすいよう、相場データは `src/data/benchmarks.json` に分離しています
- Node は 20.19 以上を推奨します
